# jbknighten-web-project

## Project 2 - HTML Foundation Page

### Description
For this project, I created a basic HTML website focused on my interest in cybersecurity. The site includes a home page and a second page connected with a navigation bar.

### What I Learned
I learned how to structure a webpage using HTML elements like headings, paragraphs, lists, tables, images, and links. I also learned how to connect multiple pages together and deploy my site using GitHub Pages.

### Challenges
One challenge I faced was getting the file paths correct between pages. I also had to fix validation errors like missing character encoding and make sure everything worked properly before submitting.
